## Project Description

PlayOn is a cutting-edge portable gaming console developed by Aerotyne Industries. This project showcases a landing page, an about page, a contact form, and an FAQ page. The website is fully responsive and uses modern web development practices including HTML5, CSS3, and Bootstrap. The project simulates a console subscription service, similar to PlayDate, allowing users to pre-order the product, learn about its specifications, and reach out via a contact form.

## Usage

The PlayOn website allows users to explore the console's features, pre-order the device, and contact the development team.

- **Home Page**: Displays the PlayOn console with a pre-order button.
- **About Page**: Provides a backstory of Aerotyne Industries.
- **FAQ Page**: Contains a table of technical specifications for the PlayOn console.
- **Contact Page**: A functioning contact form where users can send messages.

## Features

- Responsive design with Bootstrap integration.
- Hover effects for navigation and buttons.
- Functional dropdown navigation for the "About" section with links to external pages like Indiegogo.
- Custom animations on the homepage and buttons.
- Fully responsive, optimized for mobile viewing.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling, including media queries for responsiveness.
- **Bootstrap**: For responsive grid and layout management.
- **Git/GitHub**: Version control and code management.
